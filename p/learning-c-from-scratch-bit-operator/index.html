2.二进制和进制转换

2进制，8进制，10进制，16进制是数值的不同表示形式

例如：10进制下的15 对应

​ 2进制的 1111

2.1 其它进制转10进制

进制的每一位都是有权重的，比如10进制的123

3*10^0^+2*10^1^+1*10^2^ = 3+20+100=123

2进制同理

2.2 转2进制

（1）10转2

将十进制数字除2的余数倒过来

（2）八转二

从低位开始，每一位转成3个相应的二进制数即可

（3）十六转二

从低位开始，每一位转成4个相应的二进制数即可

2.3 2进制转8进制

8进制的数字范围：0~7

从低位开始，每三位为一组，转成相应的八进制数即可

2.4 2进制转16进制

从低位开始，每四位为一组，转成相应的十六进制数即可

2.5原码、反码、补码

整数的2进制表示形式有3种：原码、反码、补码

整数又分为有符号整数和无符号整数

有符号整数的原码、反码、补码的二进制表示中均由符号位和

数值位两部分组成

1.二进制的最高位（最左边）是符号位：0表示正数，1表示负数

2.正数原码，反码，补码都一样（三码合一）

3.==负数的反码=它的原码符号位不变，其它位取反（0变1,1变0）==

4.==负数的补码=它的反码+1==，负数的反码=负数的补码-1

补码转原码：-1，取反（符号位不变）注意：正数三码合一

注意

  • 0的反码，都是0

  • ==在计算机运算的时候，都是以补码的方式来运算的==

    (补码把正数和负数统一起来了)

  • ==看运算结果，要看原码==

#include <iostream>
#include <string>
#include <cmath>
using namespace std;

int main()
{
	//1.输入x，s
	 int x;
	 int ret = 0;
	 string s;
	 cin >> x;
	 cin >> s;
	
	//2.求出s的长度 
	 int n = s.size();
	 
	 //3.遍历字符串（从后往前遍历）
	 int j = 0;
	 for(int i = n - 1; i >= 0; i--)
	 {
	 		//	1)找出每一个字符-转换成整数
			//		如果是数字字符，让其减字符0；
			//		如果是字母字符，让其减字符A，再加10 
			//	2)再乘以这一位的权重
			//	3)最终求和
			if(s[i] <= '9')
				ret += (s[i] - '0') * pow(x, j);
			else
				ret += (s[i] - 'A' + 10) * pow(x, j);
			j++;
	 }
	 cout << ret << endl;
	
	return 0;
}

3.位运算操作符

2进制序列中，最高位的1位是被当做符号位

  • 符号位都是用0表示正”，用1表示负
  • 对于==整形==来说：数据存放==内存中==其实存放的是==补码==。
  • ==计算用补码，结果要看原码==（即用补码计算完后要把补码转为原码）

左移右移都不能移符号位

左移操作符（乘2的效果）

a « 1 对原变量无影响

a«= 1 左移后赋给自己

乘2的效果，主要针对正数/无符号数

右移操作符（除2的效果）

  1. 逻辑右移，左边补0

  2. 算数右移（通常是），左边原来是正数：补0

​ 负数：补1

&按位与（二进制位的运算）

对应二进制位 同为1，才为1；==有0为0==

按位或（二进制位的计算）

==有1为1==；同时为0，才为0

^按位异或（二进制位的运算）

==相同为0，相异为1==，不存在进位，不会溢出

  1. 0 ^ 任何数 = 任何数本身
  2. 自身^自身 = 0
  3. 支持交换律 3 ^ 5 ^ 3=5

~按位取反（二进制位），1个对象，0变1,1变0

4.位运算的应用

4.1判断奇数和偶数

规律1：

  • 所有偶数的2进制表示中，最低位一定是0,
  • 所有奇数的2进制表示中，最低位一定是1

规律2：

  • (x & 1) == 0，说明x是偶数
  • (x & 1) == 1，说明x是奇数

#include <cstdio>
#define READ_PERMISSION 0x01 // 0001，表⽰读权限
#define WRITE_PERMISSION 0x02 // 0010，表⽰写权限
#define EXECUTE_PERMISSION 0x04 // 0100，表⽰执⾏权限
int main()
{
    int permissions = 0x03; // ⼆进制: 0011，具有读和写权限
    if (permissions & READ_PERMISSION)
    {
    printf("阅读权限被设置\n");
    }
    else
    {
    printf("阅读权限未被设置\n");
    }
    if (permissions & WRITE_PERMISSION)
    {
    printf("写权限被设置\n");
    }
    else
    {
    printf("写权限未被设置\n");
    }
    if (permissions & EXECUTE_PERMISSION)
    {
    printf("执⾏权限被设置\n");
    }
    else
    {
    printf("执⾏权限未被设置\n");
    }
    return 0;
}

4.3获取二进制中的指定位

场景

我们希望获得一个==整数x二进制中的第i位==（其实就是确定==第i位是0还是1==），

就可以 (x » i) & 1，结果是0，那么第i位就是0，结果是1，那么第i位就是1

x » i 使得x的第i位来到最低位，然后再按位与&1就能获得第i位是1还是0

第i位是从右向左，最右边是第0位

class Solution {
public:
    uint32_t reverseBits(uint32_t n) {
    uint32_t ret = 0;
	
    uint32_t i = 0;
    for(i = 0; i < 32; i++)
    {
    	//得到n的二进制位中的第i位 
    	int b = (n >> i) & 1;
    	//移动这个二进制位到合适的位置，然后
		//存放在ret中 
    	ret |= b << (31-i);
	}
	return ret;
    }
};

应用场景

例如：在嵌入式系统中，传感器通常通过寄存器返回数据。某些寄存器可能包

含多个信息字段，例如状态标志、错误码等。这些字段往往位于寄存器的特定

位中，需要通过获取指定位的值来读取。

场景：有时候需要将一个整数x的二进制位中的，==某几位设置为1==，其余位置

class Solution {
public:
    int findComplement(int num) {
        int ret = 0;//存放的是补码
        //因为1 <= num < 2的31次方，所以num是正数
        //当num向右移到num=0的时候，有效位就处理完了
        int i = 0;//当前获取的位
        while(num)
        {
            //num的第i位是1还是0
            int b = num & 1;
            if(b == 0)
            {
                //将ret的第i位改为1
                //因为ret本来=0
                int tmp = 1;
                ret |= (tmp << i);
            }
            num >>= 1;
            i++;//num的第i位
        }
        return ret;
    }
};

4.5将指定位设置为0

例如：将x的二进制表示形式00001011中的第3位设置为0，其余位置保持不变

那么我们需要有一个数m的二进制为11110111让它按位与上x，m如何得到呢

4.6反转指定二进制位

将一个整数x的二进制中第i位取反（0变1,1变0）

x ^= (1<<i);

比如：将x的二进制表示00101011的第2位反转，其余位保持原来的值不

4.7将2进制中最右边的1置为0

例如：将整数x的二进制位00101100中的最右边的1变成0，其余位不变

就可以x &= (x-1)

这个表达式常用于计算一个二进制序列中有几个1，因为该运算每次都

会去掉一个1，一个整数的二进制中1的个数也就是该整数经过该运算后

变为0之前运算了几次的个数

int hammingWeight(int n) {

    {
        n &= (n-1);

int hammingWeight(int n) {
    int cnt = 0;
    //让n每次向右移1位，直到n为0（因为n是正数）
    while(n)
    {
        if(n & 1)
            cnt++;
        n >>= 1;
    }
    return cnt;
}

class Solution {
public:
    bool isPowerOfTwo(int n) {
        //1 - 01
        //2 - 10
        //4 - 100
        //8 - 1000
        //规律：2的幂次方的二进制表示中只有1个1
        //所以2的幂满足：1.n>=1 2.n&(n-1)==0
        return (n >= 1) && ((n & (n-1)) == 0);//(n & (n-1)可以去掉1个1
    }
};
