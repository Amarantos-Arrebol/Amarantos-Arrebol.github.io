1.6 命名空间

在我们写的第一个C++程序中有这样一段代码using namespace std;，意思是使用名字空间std

其中namespace是命名空间的关键字，std就是空间的名字

那么什么是名字空间呢？为什么要有名字空间呢？

在C++程序中，如果我们是几个人一起开发，那么可能一个人在程序中定义了一个变量，另一个

也想用这个变量名，那么就会造成命名冲突。如果我们使用命名空间，让每个人都在自己的命名空间

中写代码，相当于做了隔离，需要使用的时候，我们就指定命名空间。

所以，命名空间就是解决命名冲突的问题

std

是C++标准库的名字空间名，C++将标准库的定义实现都放到这个命名空间中，当我们需要使用

标准库中的内容时，就需要加上： using namespace std

ASCII

我们知道，在计算机中所有的数据都是以二进制的形式存储的，那么字符在内存中是

怎样以二进制方式存储的呢？我们把每个字符编一个二进制序列，这个就叫编码。

美国国家标准学会（ANSI）出台了一个标准ASCII编码

  • 字符A~Z的ASCII码值从65~90
  • 字符a~Z的ASCII码值从97~122
  • 大小写字符的ASCII码相差32
  • 数字字符0~9的ASCII码值为10
  • 换行字符\n的ASCII码值为48~57

注意：ASCII码值从0~31 这32个字符是不可打印字符，无法在屏幕上观察

变量命名的规则

  • 变量只能由数字、字母、下划线组成。
  • 变量名不能以纯数字开头，也不能包含特殊字符，如空格、连字符等
  • 变量名不能直接用语言的关键字，例如 int char 这样创建变量是非法的
  • 变量名严格区分大小写，例如myVariable和myvariable被视为两个不同的变量
  • 变量名应该具有实际意义，并且简短明了

		temp = i;
		while(temp)
		{
			sum += pow(temp % 10, 3);
			temp /= 10;
		}
		//判断是否为水仙花数
		if(sum == i) 
			cout << i << endl;
	}
	
	return 0;
}

自幂数 – 一个n位数，你的每一位的n次方之和等于你自己

3位的自幂数是水仙花数

总结做题时的注意事项

  1. 输出格式，注意许多题要求输出时中间有空格，

    有些还指定了宽度位（需要用printf）,要包含对应头文件#include <cstdio>

例如：

这道题目的输出只要求了按5位宽度输出，没要求空格

  1. 实数就是小数

  2. scanf在读的时候可以指定位数

  3. 一定要注意整数是使用int还是long long，如果是单个数字满足int，但是可能会有

    两个数相加/相乘等，那么结果可能会超出int所能表示的范围

    比如下面这道题，题目要求计算的是从1加到n的结果，需要使用long long

  4. 使用printf 时如果是long long类型的数据，要使用%lld

    long double类型的数据要使用%Lf

  5. 使用scanf时，整数使用%d，float类型的使用%f，double类型的使用%lf

    long double类型的使用%Lf

  6. 比较大的数组最好声明为全局变量，且最好多开辟一点空间

    原因：局部变量是存储在栈区的，空间有限；全局变量存储在静态区，空间较大

  7. 注意输出的标点，比如中间有逗号，但是结尾没有

    可以把第一个单独打印，后面,数字为一组

  8. int类型最大可以表示到10的9次方

  9. ==算不明白数据范围就用 long long==

  10. 注意：==结构体最后要加上分号==

  11. 注意 if() 后面不要把 == 写成了 =

  12. 子串是连续的一部分，子序列是可以不连续的

  13. 处理负数取模

    除留余数法：hash(key) = key % N

    但是需要注意，key有可能是负数，取模之后会变成负数

    • 负数补正的操作为：加上模数即可
    • 但是正数加上模数会变大，所以统一再取一次模

    最终就是**（key % N + N) % N ，简称 模加模**
