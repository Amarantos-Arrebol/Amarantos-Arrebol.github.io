1.指针

1.1内存和地址

计算机上CPU（中央处理器）在处理数据的时候，需要的数据是在内存中读取

的，处理后的数据也会放回到内存中，为了高效管理内存空间，我们需要把内

存划分成一个一个的内存单元单元，每个内存单元的大小为1个字节

一个比特位可以存储一个2进制的位1或者0

申请到的每个字节都有地址

&a取出的是a所占4个字节中地址较小的字节的地址

数值有时候也是需要存储起来，方便后期再使用。

我们把这样的地址值存放在指针变量中

1.3指针类型

指针类型的意义：1.指针的解引用 2.指针+-整数

1.3.1指针的解引用

指针的类型决定了，对指针解引用的时候有多大的权限（⼀次能操作几个字节）

比如： char* 的指针解引用就只能访问⼀个字节，而 int* 的指针的解引用就能访问四个字节

1.3.2指针+-整数

结论：指针的类型决定了指针向前或者向后走⼀步有多大（距离）。

字符的地址使用cout打印会以为是字符串，所以这里使用printf打印

2.动态内存管理

我们知道变量的创建会为变量申请一块内存空间，数组的创建其实也向内存申请一块连续的空间

这两种方式，如果创建的是全局的变量和数组，是在内存的静态区

（数据段）申请的，如果是局部变量和数组，是在内存的栈区申请的

不管是全局变量还是局部变量，申请和回收都是系统自动完成的，

不需要程序员自己处理

2.1new和delete

在C++中，使用new和delete操作符来进行动态内存管理

  • new负责申请内存
    • new操作符返回的是申请到的内存空间的起始地址，需要指针存放
    • new申请一个变量的空间，new[]申请一个数组的空间
  • delete负责释放（回收）内存
    • delete负责释放一个变量的空间，delete[]释放一个数组的空间
  • new和delete配对使用，new[]和delete[]配对使用

	return 0;
}
