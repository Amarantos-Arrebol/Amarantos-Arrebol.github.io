1、Max入门

1.1 新建patch

1.1.1新建pacth

打开max后，我们可以ctrl+n新建patch

或者也可以：file–>New Patcher

1.1.2调整界面的大小

打开patch后，在patch上面一栏是对象

我们可以先将其中的numbers拖出来看看

我们发现画面里的东西非常小，我们可以使用ctrl+鼠标滚轮，放大或缩小画面

或者点击左上角的100%符号的下拉箭头来调整画面大小

1.1.3编辑模式和工作模式

Max中有两种模式一种是编辑模式，一种是工作模式

  • 编辑模式下，我们可以新建对象，对patch里的对象做各种操作
  • 工作模式下，不能编辑物体，但是可以运行

pacth界面的左下角的有一个锁的图标，我们可以通过鼠标点击左下角的小锁，

切换patch的模式。锁打开就是编辑模式，锁关闭就是工作模式

当然也可以使用快捷键ctrl+e 来开锁或关锁

1.1.4comment注释

comment就是一个文本框，我们可以用来做注释，快捷键是c

我们可以通过调整滑块来调整comment的大小

也可以使用ctrl+j自动调整大小（该快捷键适用于所有对象）

1.1.5toggle开关

我们可以拖出来一个toggle

在建几个number对象

复制可以使用 alt+拖动对象，是新建对象

将他们连接，在锁定状态下点击toggle，我们发现是点亮的，并且所有数字框中

的数字都变为了0

当我们再次点击toggle，数字框里的数字都变回了0

1.1.6输入与输出

在Max中信号总是从outlets到inlets，一直到对象末尾

  • inlets就是object inputs对象输入
  • outlets就是object outputs对象输出

1.2 Matrices和Texture

  1. 什么是CPU和GPU

    • textures使用GPU工作，matrices使用CPU工作

      • CPU是核心处理，GPU是图形处理单元

        CPU主要用于处理复杂的计算，但不是高度并行的

        GPU是高度并行的，但不执行真正复杂的操作

并行：

  • 比如CPU可能只有8个core，那么CPU处理器在每个计算机周期只能同时

    发生8个并行操作。

  • 但是GPU是由很多core组成的，这些cores都是并行执行大量操作的

    例如：在屏幕上绘制像素，因为我们有很多像素，屏幕的每次刷新率都

    必须将这些像素调整为新值，这意味着计算机要处理大量信息，因而需要

    许多核心同时处理

  1. 在Max中使用GPU的强大功能来实现视觉效果

    • jitter Matrix停留在我们计算机的随机存取存储器RAM中

      jit.movie发送的matrix信息是jit_matrix加上矩阵的名称

      然后每个接收到这个信息的对象就会去计算机的RAM中去找

    • Textures 工作需要OpenGL Context

      当我们想向GPU发送指令时，我们需要使用类似opengl的东西

      OpenGL是一组与GPU通信的命令，也被称为应用程序编程接口即API

      因此Max使用OpenGL库或者说API

      创建这个context我们只需要创建一个jit world对象

      有了这个对象之后，我们可以用max中的所有GPU对象

      激活这个对象需要使用toggle，此后，我们不再需要使用bang就可以连续播放影片

      texture不在RAM中，而是在显存vRAM（video random axis memory中，速度更快

      • 如果用矩阵的方式播放，视频数据需要从主内存（RAM）传输到显存（VRAM）才能被 GPU 处理。
      • 如果用纹理的方式播放，数据已经在显存中，省去了传输过程，大幅提升了性能。

1.1 初识Max/MSP/jitter

  • 如何创建对象：n 或 双击空白处

    1、jit.pwindow对象 ：屏幕，可以将视频显示到屏幕上进行预览

    2、jit.movie对象

    3、画框对象：lcd

    4、合成list的工具的对象：pak 后面可以加 s 或 0 或 0.等

    5、拆解类工具：unpack

  • 如何创建信息框：m

    信息框里输入的数据，常用于给不同对象下达指令

    1、信息框中输入read指令，表示读取。可以是读取字符，数字，也可以读取音视频等

    ​ 当连在jit.movie对象上时，点击red信息框可以读取电脑上的视频/音频等

    2、设置画笔颜色的信息：frgb 255 0 0 代表红色

    3、画椭圆：paintoval 50 50 100 100

    4、画矩形：paintrect 50 50 100 100

  • 如何创建数字框

    创建整数框：i

    创建小数框：f

  • 触发指令：b

    会生成一个bang指令，让对象执行一次

1.2程序的运行方向

横向排列运行规则：max遵循从右向左运行的原则

纵向排列运行规则：max遵循从下往上运行的原则

斜向排列运行规则：max优先遵循从右向左运行的原则
