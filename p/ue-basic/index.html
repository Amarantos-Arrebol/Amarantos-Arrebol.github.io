【虚幻引擎】-ue基础入门

2025-11-02

🥳1. 创建项目全流程

🍋‍1.1 创建自己的工程

  • 打开对应版本的虚幻引擎

    • 注意：高版本的可以打开低版本的项目，但低版本无法打开低版本的项目

  • 在虚幻项目浏览器中选择是打开已有项目还是新建项目

    • 只能打开对应版本的项目

      • 比如：如果启动的是4.22版本的虚幻引擎，那么就只能打开版本是4.22的项目

    • 我们可以按需要选择新项目的类型（以游戏为例）

  • 选择模版

  • 项目设置

    • C++项目还是蓝图项目？

      • 蓝图项目：使用蓝图脚本实现功能（不碰任何一行代码，底层还是C++）
      • C++项目：使用C++代码实现功能（需要电脑上装有VS）

    • 目标硬件（PC或移动设备）

      • 选择PC

    • 目标平台图像画质

      • 选择最高质量

    • 是否包含初学者内容包

      • 选择包含

    • 选择项目创建路径以及项目名称

      • 项目名称开头只支持字母和中文
      • 项目名称支持的特殊符号：_

  • 点击创建项目

🍇1.2 虚幻4的工程文件夹结构

  • Config
    • 包含了项目设置，键盘输入之类的配置文件
  • Content
    • 存放引擎或游戏的内容，包括地图、贴图、模版、材质、蓝图等
  • Saved
    • 包含自动保存内容，配置(*.ini)文件以及日志文件
  • .uproject
    • 项目启动程序，开启我们的项目

🫐1.3 运行我们的游戏

  • 在内容浏览器找到文件夹名字带有BP的文件夹
  • 打开Map文件夹中的关卡
  • 点击工具栏中的播放图标开始游戏
    • 补充：播放游戏时切换到/取消全屏，F11

🍉1.4 初学者内容包

  • 打开 内容浏览器中的StartContent 文件

    • 里面是官方提供给初学者使用的素材包

    • 大致包括建筑模型，音频，蓝图，HDRI，关卡，材质，粒子，道具，贴图等

🍍1.5 快速添加功能或内容包

如果在创建项目的时候，没有选择包含初学者内容包，那么可以通过以下步骤快速添加初学者内容包

  • 在内容浏览器中的添加/导入中选择添加功能或者内容包

  • 选择内容包，可以将初学者内容包添加到项目

除了添加初学者内容包，我们还可以添加蓝图功能，C++功能等

🤗2.视口导航

在视口中流畅地移动视角的方法

  • 方法一：只用鼠标

    • 按住鼠标左键

      • 前后移动鼠标：实现视口的前进/后退
      • 左右移动鼠标：实现视口的旋转

    • 按住鼠标右键

      • 前后移动鼠标：实现视口的俯/仰
      • 左右移动鼠标：实现视口的旋转

    • 按住鼠标中间

      • 前后移动鼠标：实现上下移动视口

      • 左右移动鼠标：实现视口的左右移动

  • 方法二：使用键盘的WASD（前提：按住鼠标的右键）

    • W：前进 S：后退 A：左移 D：右移

    • Q：镜头的下降 E：镜头的上升

    • Z：镜头拉远（会自动复原） C：镜头拉近（会自动复原）

  • 其它视口导航操作

    • 快速聚焦到物体身上
      • 先选中要聚焦的物体，再按住F
      • 双击世界大纲上的模型
    • 视口的旋转：alt+鼠标左键
    • 视口的缩放：alt+鼠标右键
    • 视口的平移：alt+鼠标中键

  • 编辑

❄️3.2 关卡编辑器(Level Editor)工具栏

  • 保存当前关卡

    • 一般不使用这个，一般使用下方的保存所有

  • 源码管理

    • 如果有很多个人同时开发这个项目，那么就需要版本的管理

  • 内容

    • 如果不小心关掉了内容浏览器的窗口，就可以点击内容重新打开内容管理器

  • 虚幻商场

    • 可以链接到虚幻商城

  • 设置

    • 项目设置
    • 插件
    • 一些物体的选择（比如按T，就可以选择半透明物体）
    • 扩展性的设置（比如改画质）
    • 预览之后要打包平台的效果

  • Magascans

    • Bridge的插件

  • 蓝图

    • 打开蓝图类，打开关卡蓝图等

  • 过场动画

    • 可以添加一些镜头，给我们的场景进行介绍

  • 构建

    • 当前物体被移动后，阴影就不正确了，会出现提示例如：光照需要构建

      • 这时我们需要点击构造或仅构建光照让提示消失
      • 在构建的时候我们还可以选择下方的光照质量

    • 还可以构建反射、几何体、路径等

  • 启动

    • 可以在不同的平台来启动我们的程序

运行（测试关卡）

  • 选中的视口

    • 默认在当前视口进行播放

  • 移动设备预览

    • 会跳出一个移动端的窗口，来预览移动设备上的效果

  • 新建编辑器窗口

    • 新建一个视口来运行

  • 独立进程游戏

    • 会单独跳出一个新窗口来运行

  • 模拟

    • 跳出玩家控制，以上帝视角来查看整个关卡
    • 点击播放后，旁边有一个拥有的选项，点击可以切回玩家控制

  • 高级设置

    • 可以调整播放时游戏视口的分辨率

🌊3.3 模式(Modes)面板

显示各种编辑模式，通过该面板可以使用特定的编辑界面来编辑不同类型的Actor

  • 选择

    • 可以将下面图标里的物体通过鼠标左键拖动到我们的场景中

    • 基本：一些基本的几何体和触发器

    • 光源

    • 过场动画

      • 可以选择一些专业的摄像机

    • 视觉效果

      • 一些大气、雾、云、反射、捕捉、后期处理体积等
      • 这些效果拉到场景中可以实时看到效果

    • 几何体

      • 建模的工具

    • 体积

      • 一些不同的容器，比如触发体积（空气墙），可以阻挡角色，不让角色过去

    • 所有类

  • 地形

    • 可以搭建我们的开发世界
    • 绘制/雕刻地形
    • 在地形上弄一些河流，道路
    • 通过导入一些高度图，快速生成地形

  • 植物

    • 可以在我们已经搭建好的地形上，刷一些树、草、灌木、石头等模型，来美化我们的模型

  • 笔刷编辑

    • 编辑几何体，例如改变几何体形状

  • 网格体绘制

    • 在一个面片/模型上刷上我们想要的材质

☄️3.4 内容浏览器面板

内容浏览器是在虚幻编辑器中创建、导入、组织、查看和修改内容资源的主要区域

它可以管理内容文件夹，在资源上执行其它实用操作（如重命名、移动、复制和查看引用）

内容浏览器可以搜索游戏中的所有资源并与之交互

注：资源被放置到关卡中后，就会被称之为Actor

  • 添加/导入

    • 可以快速添加一些新的类、功能、内容包等

    • 右击空白区域也可出现

    • 这里的导入与文件中的导入到关卡是不同的，一般使用这里的导入

  • 保存所有

    • 快捷键：ctrl+shift+s

  • 文件夹的显示/隐藏，搜索

  • 过滤器

    • 当我们的项目越来越庞大，里面的东西越来越多时，可以通过过滤器来筛选

  • 移动或复制文件

    • 可以通过拖动某个文件夹到另一个文件夹中，选择移动/复制/拷贝

    • 还可以将文件夹拖到左边的文件目录中的某个文件夹下

  • 视图选项卡

    • 例如：选择显示引擎内容，就可以查看虚幻引擎自带的一些素材
    • 缩略图：可以控制文件夹的缩放，同样，我们也可以使用ctrl+鼠标滚轮来进行缩放

🌟3.5 视口面板

视口就是查看你在虚幻引擎中创建的世界场景的窗口

你可以像在游戏中导航那样来导航视口，或者可以像使用建筑蓝图进行方案设计那样来应用视口

虚幻编辑器视口包含各种工具和查看器，可以帮助你精确地看到所需数据

透视视口：使用消失点呈现出三维世界场景 — 正常视角模式

正交视口：将世界场景呈现为二维示意图

  • 我们可以在窗口处打开多个视口，提高工作效率

  • 视口选项

    • 点击下拉的小箭头

    • 显示数据，显示FPS帧率

    • 游戏视图：会隐藏我们在游戏中看不到的东西

    • 沉浸模式F11：让我们的视口全屏 shift+F11：全屏虚幻引擎

    • 书签，保存某个视角，快捷键ctrl+0到9，即可以保存9个书签

      输入对应的数字即可回到保存书签的那个视角

      注意：书签可以被覆盖

    • 在此处创建相机

    • 高分辨率截图

      • 截图尺寸系数，与电脑的性能有关，一般2~3

    • 布局：可以选择视口的不同模式

  • 视口类型

    • 透视视口，顶视口，底视口，左视口，右视口，前视口，后视口

  • 视口模式

    • alt+0，alt+2到8可以切换不同的通道来显示

    • 在缓冲显示下的总览中，我们可以看到不同通道下的效果

  • 显示

    • 里面是一些标签，如果不勾选，那么编辑的时候就会被隐藏起来

      播放的时候还是会显示

🌌3.6 世界大纲视图面板

世界大纲面板以层次化的树状图形式显示了当前关卡中的所有Actor

⚡3.7 细节面板

当我们点击场景中的一个模型/物体后，细节面板就会显示当前物体的一些具体的参数

我们可以在细节面板中对当前模型进行一些调整，比如改变它的位置，改变它的模型

替换它的材质，更改它的碰撞，更改它的光照模型，调整它的渲染距离等

😁4.物体编辑（坐标系，变换单位）

  • 坐标系

    • Z轴：上下方向 ，Y轴：左右方向 ，X轴：前后方向

  • 处理变换

    • 移动：W

    • 缩放：R

    • 旋转：E

    • 还可以在右边的细节面板中手动地输入精确的数值，让该模型移动到对应的位置等

  • 世界坐标轴和局部坐标轴

    • 当我们对物体进行旋转后，它自己的坐标轴就会发生改变

    • 世界坐标轴永远保持水平

      • 可以点击视口右上角的地球图标，使物体的坐标轴变为世界坐标轴

  • 单位和度量

    • 虚幻引擎的单位是uu，相当于厘米
    • 默认情况下玩家通常在游戏场景中高约6英尺，相当于180cm或180uu

  • 对齐到网格

    • 可以设置每次移动多少厘米，比如每次移动10厘米

      • 选择视口右上方的设置位置网格对齐值

    • 还可以设置每次旋转的度数

      • 选择视口右上方的设置旋转网格对齐值

    • 注意：每一个都要先打开它们的开关

  • 调整摄像机的速度

  • 枢轴点

    • 我们的模型导入到虚幻引擎中都会有一个坐标轴，有时候这个坐标轴会跑地很远

    • 将坐标轴重新放到物体上的步骤

      • 点击坐标轴中间的白色圆点，可以对坐标轴进行移动，将其移动到模型上

      • 右击白色的点，找到锚点中的设置为锚点偏移量

      • 这种方式只适用于当前关卡

  • 在场景中快速选取很多个模型

    • 在视口中框选模型，框选的快捷键：ctrl+alt+鼠标左键

  • 复制的快捷键：选中物体，按住alt，再拖动

  • 物体在内容浏览器中显示：ctrl+b

😚5.导入资源，项目迁移和管理

5.1 导入资源

  • 文件类型

    UE4支持导入各种类型文件的内容

    例如：将模型的贴图、模型、音频等导入UE4中

    • 一些常用的文件类型和资源类型以及相关的应用程序

    • 许多资源不是导入的，而是直接在编辑器中创建的

    ​ 例如：蓝图类、粒子系统，材质和材质实例等

  • 在将内容导入到引擎前，我们需要对外部文件进行合理地组织，并遵循命名规范

  • 内容浏览器

    • 导入外部应用程序中所创建的外部内容的两种常用方法

      • 方式一：使用内容浏览器，单击导入按钮或单击资源视图并选择导入到

      • 转到操作系统的文件管理器，选择要导入的文件，然后将其拖放到内容浏览器中

  • 资源图标

    • 资源图标左下角的小星号表示资源尚未保存

    • 按ctrl+s保存所有资源

  • 引用查看器

    大多数的资源都依赖一个或多个其它资源

    • 例如，如果在静态网格体编辑器中将一个材质分配给静态网格体资源，那么静态网格体就会引用该材质
    • 这意味着，当加载静态网格体资源时，还需加载引用的材质
    • 相应地，材质也将加载材质编辑器中引用的纹理

    右键我们要查看的资源，右击选择引用查看器

5.2 项目迁移

  • 项目迁移的两种情况

    • 情况一：

      我们电脑上通常有两个项目，一个是新建的项目，一个是过去以往的项目

      当我们新建一个项目的时候，可能会用到以前做过的项目的一些素材

      这个时候我们就需要迁移，把旧的工程里的素材迁移到新的项目中

    • 情况二：

      分享给同事一些自己的东西，就需要迁移项目，把同事可能需要或你想要分享的

      迁移到新的项目中去。或者老板验收，需要弄到一个干净的项目里再打包给你的老板

  • 迁移文件的步骤

    • 右键要迁移的文件，选择迁移

    • 选择要迁移的内容，选择完后点击确定

    • 选择要迁移到的路径，一定要迁移到目标项目的Content文件夹下面

      如果同时选择多个蓝图/模型/粒子特效来进行迁移，迁移选项就在资产操作中

  • 网上下载的如何迁移

    • 从网上下载的资源中找到Content目录下的资源，复制到目标项目的Content文件夹下
    • 最好新建一个文件夹存放网上下载的资源中Content目录下的资源，再进行复制

  • 打包项目回家的步骤**

    • 找到创建项目的根目录
    • 将缓存文件（Intermediate）和自动保存文件（Saved）删除
    • 右键最外面的项目，添加到压缩文件

5.3 项目管理

😎6.三维模型软件导入虚幻4

maya的插件：虚幻商场的maya livelink，能动画同步

3dmax和C4d的插件：DataSmith

文件格式的用途

  • .fbx 和 .obj的格式都是三维通用模型格式，都可以用在目前几乎所有主流的三维软件中

  • .fbx 中包含动画、材质特性、贴图、骨骼动画、灯光、摄像机等信息

    .obj 中不包含

  • .abc 格式，支持动画、粒子等，烘焙三维场景的模型、流体、动画、特效等数据

总结

  • 静态或人物骨骼模型，我们通常会使用 .fbx
  • 模型导入到虚幻引擎中有两个类别
    • 静态网格体：静态的摆件，例如家居，道具，鼠标，椅子，房子等
    • 骨骼网格体：有动画的，例如人物模型（要跑步、走路），机械臂

导入流程

  • 用这几个三维模型软件统一导出一个通用的模型的格式（.fbx、.obj、.abc）

    • 导出时带贴图：maya在导出时要勾选嵌入式媒体

  • 导入到虚幻引擎，两种方式

    • 方式一：选择导入到/Game/_Orion/Meshes

    • 方式二：直接拖拽到虚幻引擎中

  • 导入后的设置

    • Mesh中，如果是静态的模型，他就会自动帮你生成骨骼

      • LOD就是能为这个静态网格体自动生成多细节层次

      • 合并网格体，如果勾选，我们的模型就不是一个一个的，而是一个整体

      • 法线导入的方法，如果导入的模型出现了问题，例如出现了黑边，就可以

        更改法线导入的方法

      • 法线生成的方法，如果选择Mikk TSpace就可能出现一些黄色的警告，

        如果你的模型出现了问题，可以把法线生成方法改成内置

      • Miscellaneous轴的设置

        • 转换场景：就是将场景从fbx坐标系转换到虚幻引擎的坐标系

        • 强制前x轴：强制将这个模型的坐标系为前X轴而不是负Y轴

        • 一般是让模型的坐标系和UE4里的坐标相匹配，UE4中的坐标系是Z轴朝上

          而maya中是Y轴朝上

        • 转换场景单元：虚幻引擎的单位默认是厘米，如果别的软件里默认单位不是

          厘米，那么导入进来模型的大小就会发生变化

    • 如果勾选骨骼网格体，那么导入的模型就会是一个骨骼模型就会有动画

      • 使用TO作为参考姿势：就是使用动画的第0帧作为参考姿势

        如果不勾选，那么导入进来的模型可能会变形和一些拉伸

      • 导入变形目标，常用于面部表情

        勾选后，我们导入一些顶点变形动画就能导入进来，然后播放对应的动画

      • Animation，我们导入骨架网格体的时候，是否要导入动画

        因为你导入的这个骨架网格体模型文件里可能携带有动画，这时就需要去勾选动画

      • 虚幻引擎里默认的动画帧率就是30帧
