1.2.3变量和数据类型

变量和常量

变量声明的基本格式：类型 变量名;

声明变量的方法：

  1. 只声明变量，例：int a;

  2. 声明变量的同时初始化该变量，例：int b = 1;

注意：变量要先声明，后使用

变量命名规则

变量–首字母大写

函数、类–所有单词首字母都大写

常量

const常量声明的基本格式：const 变量类型 变量名 = 初始值

例如：const int a = 10;

常量的特点：

  1. 常量必须初始化
  2. 常量不能被修改

作用：声明一些游戏中不会发生变化的变量（比如：PI、角度转弧度、弧度转角度、重力加速度等）

const定义的常量是在编译的时候就确定了的，而readonly定义的常量是在运行时才确定的。

readonly常量不能定义在某一个方法中，只能定义在类体中

数据类型

基本数据类型大致可分为三类：有符号整型，无符号整型，浮点型(小数)，特殊类型

  • 有符号整型：byte、ushort、uint、ulong
  • 无符号整型：sbyte、short、int、long
  • 浮点型(小数)：float、double、decimal
  • 特殊类型：bool、char、string

1.2.4异常捕获🌟

异常处理就是当异常发生时，对异常处理的方式。

基本语法

try{
	//代码/可能有异常
}catch(Exception e){
	//捕获到异常
	//1.当异常发生时
	//2.系统将异常封装成Exception对象e，传递给catch
	//3得到异常对象后，程序员，自己处理
	//4.注意，如果没有发生异常catch代码块不执行
finally{
	//1.不管try代码块是否有异常发生，始终要执行finally
	//2.所以，通常将释放资源的代码，放在finally
}

注意：

  1. 异常如果没有处理，会直接退出程序，如果捕获了异常，则程序会继续执行剩下的代码
  2. 如果try块中发生异常，则try块剩下的代码不会执行，直接进入catch中
  3. finally中的代码一定会执行！！！不管是否发生了异常

            //隐藏光标
            Console.CursorVisible = false;

            int x = 0, y = 0;
            while (true)
            {
                //清空之前信息的方式1
                //Console.Clear();

                //设置光标的位置
                Console.SetCursorPosition(x, y);
                Console.Write("■");

                //得到玩家输入
                char input = Console.ReadKey(true).KeyChar;

                //清空之前信息的方式2
                Console.SetCursorPosition(x, y); // 光标放到之前的位置
                Console.Write("　");//用空格把之前的方块擦掉

                //根据玩家的输入，设置光标坐标
                switch (input)
                {
                    case 'W':
                    case 'w':
                        if (y - 1 >= 0) y--;
                        Console.SetCursorPosition(x, y);
                        break;
                    case 'A':
                    case 'a':
                        if (x - 2 >= 0) x-=2; //  y的距离视觉上是x的两倍
                        break;
                    case 'S':
                    case 's':
                        if (y + 1 < Console.BufferHeight) y++;
                        break;
                    case 'D':
                    case 'd':
                        if (x + 2 < Console.BufferWidth) x+=2;
                        break;
                }
            }
        }
    }

                    case 'S':

                            // 关闭控制台
                            Environment.Exit(0); 
                        }
                        break;
                }

                if(isQuitWhile == true)
                {
                    break;
                }
            }

                    default:
                        break;
                }
                Console.WriteLine("你选择了“"+ sexStr + occupation +"”，攻击力：" + atk + "，防御力：" + def + "，职业技能：" + skill);
            }
            catch
            {
                Console.WriteLine("请输入数字");
            }
        }
